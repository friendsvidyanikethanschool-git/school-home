Connect to GitHub

This page edits content.json and notices.json directly in your repository. Nothing here changes index.html.

You'll need a GitHub Personal Access Token with write access to this repository. It's stored only for this browser tab (cleared when you close it) — never saved, never sent anywhere except GitHub's API.
How do I get a token?

On GitHub: click your profile photo → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Set "Repository access" to this repo only, and under Permissions grant Contents: Read and write. Copy the token — GitHub only shows it once.
